# 促銷圖片模板

這個目錄包含 Chrome Web Store 促銷圖片的 HTML 模板。

## 📐 圖片規格

### 必需圖片
- **截圖（Screenshots）**: 1280x800 或 640x400 px
- **小型宣傳圖（Small Tile）**: 440x280 px

### 推薦圖片
- **大型橫幅（Marquee）**: 1400x560 px

## 📄 模板文件

### 1. marquee-1400x560.html
**大型橫幅圖（1400 x 560 px）**
- 最醒目的宣傳位置
- 展示主要功能和價值主張
- 包含動畫效果的節拍指示器

**一鍵截圖步驟（推薦）：**
1. 用 Chrome 瀏覽器開啟此文件
2. 點擊頂部綠色大按鈕「📸 點擊截圖並下載」
3. 等待 1-2 秒，圖片會自動下載到下載資料夾
4. 完成！圖片已經是精確的 1400 x 560 px 尺寸

## 🎨 配色方案

所有模板使用與擴充功能一致的配色：

- **主色調**: 紫色漸層 (#667eea → #764ba2)
- **開始按鈕**: 綠色漸層 (#4CAF50 → #45a049)
- **暫停按鈕**: 橙色漸層 (#FF9800 → #F57C00)
- **停止按鈕**: 紅色漸層 (#f44336 → #d32f2f)
- **背景**: 淺灰色 (#f8f9fa)

## 💡 一鍵截圖功能說明

### 特色功能
✅ **大型綠色按鈕** - 頂部中央，24px 字體，易於辨識
✅ **自動精確尺寸** - 使用 html2canvas 庫，確保尺寸完全正確
✅ **自動下載** - 點擊後自動下載 PNG 圖片到下載資料夾
✅ **檔名含時間戳** - 自動命名為 `slow-jogging-timer-XXX-YYYY-MM-DD.png`
✅ **視覺反饋** - 按鈕文字會顯示處理狀態（截圖中...→完成）
✅ **無障礙設計** - 特別為視力較弱的用戶優化

### 使用步驟
1. 用瀏覽器開啟 HTML 文件
2. 點擊綠色大按鈕
3. 等待 1-2 秒
4. 圖片自動下載完成

### 如果一鍵截圖失敗
如果自動截圖功能無法使用，可以使用以下傳統方法：

#### Windows
- **Snipping Tool** (內建)
- **Snip & Sketch** (Win + Shift + S)
- **ShareX** (免費開源)

#### macOS
- **Screenshot** (Cmd + Shift + 4)
- **Cleanshot X** (付費，功能強大)

#### 跨平台
- **Firefox Screenshots** (瀏覽器擴充)
- **Chrome DevTools** (F12 → Device Toolbar → 調整尺寸)

## 📝 進階技巧

### 使用 Chrome DevTools 精確截圖

1. 開啟文件後按 F12 打開開發者工具
2. 按 Ctrl+Shift+M (Windows) 或 Cmd+Shift+M (Mac) 進入裝置模擬模式
3. 在頂部選擇「Responsive」
4. 輸入精確尺寸（例如：1400 x 560）
5. 點擊右上角三點選單 → "Capture screenshot"
6. 自動保存為精確尺寸的截圖

### 批次處理

如果需要多語言版本：
1. 修改 HTML 文件中的文字內容
2. 保存為不同檔名（例如：marquee-en.html）
3. 依序截圖

## ✅ 提交檢查清單

提交到 Chrome Web Store 前，確保：

- [ ] 所有圖片尺寸正確
- [ ] 圖片格式為 PNG 或 JPG
- [ ] 圖片檔案大小 < 5MB
- [ ] 圖片清晰，無模糊或失真
- [ ] 文字清晰可讀
- [ ] 至少準備 1 張截圖
- [ ] 建議準備 3-5 張展示不同功能的截圖

## 🌐 多語言版本

如需創建其他語言版本：
1. 複製現有 HTML 文件
2. 修改文字內容為目標語言
3. 根據語言調整字體大小（某些語言可能需要更大字體）
4. 保持視覺風格一致
