# 促銷圖片模板

這個目錄包含 Chrome Web Store 促銷圖片的 HTML 模板。

## 📐 圖片規格

### 必需圖片
- **截圖（Screenshots）**: 1280x800 或 640x400 px
- **小型宣傳圖（Small Tile）**: 440x280 px

### 推薦圖片
- **大型橫幅（Marquee）**: 1400x560 px

## 📄 模板文件

### 1. marquee-1400x560.html
**大型橫幅圖（1400 x 560 px）**
- 最醒目的宣傳位置
- 展示主要功能和價值主張
- 包含動畫效果的節拍指示器

**截圖步驟：**
1. 用 Chrome 瀏覽器開啟此文件
2. 按 F11 進入全螢幕模式（或 Fn+F11）
3. 縮放至 100%（Ctrl+0 或 Cmd+0）
4. 使用截圖工具截取整個橫幅區域
5. 確保尺寸為 1400 x 560 像素

### 2. small-tile-440x280.html
**小型宣傳圖（440 x 280 px）**
- 用於商店列表顯示
- 簡潔明瞭的設計
- 突出核心功能

**截圖步驟：**
1. 用 Chrome 瀏覽器開啟此文件
2. 縮放至 100%
3. 截取整個方塊區域
4. 確保尺寸為 440 x 280 像素

### 3. screenshot-feature-1.html
**功能展示截圖（1280 x 800 px）**
- 展示 BPM 控制功能
- 左右分欄設計：功能說明 + 實際界面
- 適合作為商店頁面的功能截圖

**截圖步驟：**
1. 用 Chrome 瀏覽器開啟此文件
2. 縮放至 100%
3. 截取整個內容區域
4. 確保尺寸為 1280 x 800 像素

## 🎨 配色方案

所有模板使用與擴充功能一致的配色：

- **主色調**: 紫色漸層 (#667eea → #764ba2)
- **開始按鈕**: 綠色漸層 (#4CAF50 → #45a049)
- **暫停按鈕**: 橙色漸層 (#FF9800 → #F57C00)
- **停止按鈕**: 紅色漸層 (#f44336 → #d32f2f)
- **背景**: 淺灰色 (#f8f9fa)

## 💡 截圖工具推薦

### Windows
- **Snipping Tool** (內建)
- **Snip & Sketch** (Win + Shift + S)
- **ShareX** (免費開源)

### macOS
- **Screenshot** (Cmd + Shift + 4)
- **Cleanshot X** (付費，功能強大)

### 跨平台
- **Firefox Screenshots** (瀏覽器擴充)
- **Chrome DevTools** (F12 → Device Toolbar → 調整尺寸)

## 📝 進階技巧

### 使用 Chrome DevTools 精確截圖

1. 開啟文件後按 F12 打開開發者工具
2. 按 Ctrl+Shift+M (Windows) 或 Cmd+Shift+M (Mac) 進入裝置模擬模式
3. 在頂部選擇「Responsive」
4. 輸入精確尺寸（例如：1400 x 560）
5. 點擊右上角三點選單 → "Capture screenshot"
6. 自動保存為精確尺寸的截圖

### 批次處理

如果需要多語言版本：
1. 修改 HTML 文件中的文字內容
2. 保存為不同檔名（例如：marquee-en.html）
3. 依序截圖

## ✅ 提交檢查清單

提交到 Chrome Web Store 前，確保：

- [ ] 所有圖片尺寸正確
- [ ] 圖片格式為 PNG 或 JPG
- [ ] 圖片檔案大小 < 5MB
- [ ] 圖片清晰，無模糊或失真
- [ ] 文字清晰可讀
- [ ] 至少準備 1 張截圖
- [ ] 建議準備 3-5 張展示不同功能的截圖

## 🌐 多語言版本

如需創建其他語言版本：
1. 複製現有 HTML 文件
2. 修改文字內容為目標語言
3. 根據語言調整字體大小（某些語言可能需要更大字體）
4. 保持視覺風格一致
